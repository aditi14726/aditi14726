## 💼 Featured Projects

### 🛡️ Real-Time Phishing URL Detector & Browser Sandbox
> **Tech Stack:** Python, XGBoost, Flask REST API, JavaScript (Chrome Manifest V3 Architecture), DNS Protocols.

*   **Business Problem:** Mitigating social engineering vectors by blocking inline malicious URLs at the browser client before resource execution.
*   **Architecture & Engineering:** Engineered a **13-feature extraction pipeline** evaluating URL structural anomalies, domain age telemetry, and DNS authoritative records. Integrated an explainability layer showing top contributing vectors per verdict.
*   **Production Highlight:** Exposed an asynchronous Flask REST API enabling sub-100ms real-time lookups inside a Chrome Extension.

### 🧵 Concurrent Resource Allocation Engine (Banker's Algorithm)
> **Tech Stack:** React.js, Tailwind CSS, JavaScript (ES6+), Python OS Simulation Layer.

*   **Business Problem:** Visualizing and testing safe allocation matrices for concurrent processes contesting for finite shared resources.
*   **Architecture & Engineering:** Designed a stateful matrix computation engine evaluating vector requests against safety thresholds to prevent resource starvation and cyclic deadlocks.
*   **Production Highlight:** Completely interactive UI rendering state mutations across allocation tables in real-time.

### ⚛️ Mathematical Quantum Tic-Tac-Toe Engine
> **Tech Stack:** Python Core, Tkinter Engine, Matrix Dependency State Mapping.

*   **Business Problem:** Simulating complex quantum mechanics (superposition, multi-state entanglement, cyclical collapse vectors) inside a traditional game loop framework.
*   **Architecture & Engineering:** Developed a graph-based cyclic check algorithm that detects quantum entanglement loops and executes deterministic state collapses.
*   **Production Highlight:** Presented at the UPES University Open Day; runs independently with zero external multi-threading/heavy quantum dependencies.
